# Cloud Cost Optimizer

A Python and Machine Learning project that analyzes cloud usage data, predicts cloud costs, and provides cost-optimization recommendations.

## Project Objective

The aim of this project is to help organizations understand cloud spending patterns, predict future cloud costs, and identify possible ways to reduce unnecessary expenses.

## Features

- Cloud cost dataset analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Cloud cost visualizations
- Machine Learning cost prediction
- AI-based cost optimization recommendations
- FinOps report generation

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

1. Dataset loading
2. Data analysis
3. Data cleaning
4. Data visualization
5. Machine Learning model training
6. AI cost optimization recommendations
7. FinOps report generation

## Project Files

| File | Description |
|---|---|
| `phase1_dataset.py` | Loads and understands the cloud cost dataset |
| `phase2_analysis.py` | Performs initial data analysis |
| `phase3_cleaning.py` | Cleans and preprocesses the dataset |
| `phase4_visualization.py` | Creates cloud cost visualizations |
| `phase5_machine_learning.py` | Builds and evaluates the ML prediction model |
| `phase6_ai_recommendation.py` | Generates cloud cost optimization recommendations |
| `phase7_finops_report.py` | Generates the final FinOps report |
| `main.py` | Main file for running the project |
| `gcp_final_approved_dataset.csv` | Dataset used in this project |

## How to Run

Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

Run the project:

```bash
python main.py
```

## Model Evaluation Metrics

The Machine Learning model is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
